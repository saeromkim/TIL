#### 2. 생성자 함수 만들기 예제
```
function Product(name, weight, price){
	this.name = name;
	this.weight = weight;
	this.price = price;
};

Product.prototype.print=function(){
	this.calculate=this.weight * this.price;
	console.log(`${this.calculate}원`);
};

let products=new Product("돼지삼겹살",100,1690);


products.print();
```
결과
```
 >169000원
```
