#### 문제 설명
자연수 N이 주어지면, N의 각 자릿수의 합을 구해서 return 하는 solution 함수를 만들어 주세요.  
예를들어 N = 123이면 1 + 2 + 3 = 6을 return 하면 됩니다.

#### 제한사항
N의 범위 : 100,000,000 이하의 자연수
#### 입출력 예
N  	answer  
123  	6  
987  	24  

9 + 8 + 7 = 24이므로 24를 return 하면 됩니다.

### 문제 풀이 (java)

```
import java.util.*;

public class Solution {
    public int solution(int n) {
        int sum = 0;
        
        while(n>0){
            sum +=n%10;
            n/=10;
        }

        return sum;
    }
}
```

### 문제 해설

먼저 자연수 1의 자릿수를 구한다.  
만약 123 숫자가 주어졌을 때 `123 % 10 = 3` 이다.  
그리고 나누기 10을 해서 각 자릿수를 오른쪽으로 한칸씩 이동한다. `123/10=12`  
그리고 다시 `12%10=2` > `12/10=1` > `1%10=1` > `1/10=0` 을 더이상 이동시킬 자리가 없을때까지 반복한다.
